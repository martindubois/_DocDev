Les noms de projet dans «HCK Studio» doivent respecter la nomenclature
«Client-Projet-Version-Bits-Index» ou
    Client  = Le code associe au client;
    Projet  = Le numéro de projet (3 chiffres);
    Version = l'abréviation du nom de la version de Windows visée
        7  = Windows 7;
        81 = Windows 8.1;
        10 = Windows 10.
    Bits    = le nombre de bits de la plateforme visée, soit 32 ou 64;
    Index   = une lettre simplement incrémentée à chaque essai.
