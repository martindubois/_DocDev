Les nom de projet dans "HCK Studio" doivent respecter la nomenclature
"Projet-Version-Bits-Index" ou
	Projet	= l'abreviation du nom de projet visant une certification (maximum
		  de 4 lettres)
	Version	= l'abreviation du nom de la version de Windows visé
		  81 = Windows 8.1
	Bits	= le nombre de bits de la plateforme visé, soit 32 ou 64
	Index	= une lettre simplement incrémenté à chaque essaie
